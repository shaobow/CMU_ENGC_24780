
Use these keys on the graphics window:  
	Click the left mouse key to select current gear 
	A : Add a gear from input file 
	D : Delete the current selected gear from the collection 
	L : List all the data for the currently selected gear on the console window 
	E : Edit the current gear 
	T : Attach the current gear to another selected gear 
	M : Mesh the current gear into another selected gear 
	S : Save all gears to a single file 
	Up/Down arrows: select current gear in the collection 
	Shift + Up/Down arrows: select the other gear in the collection 
	Left/Right arrows: rotate the currently selected gear 
	X/esc : eXit the program
